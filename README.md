### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/GermanAlexis/SerempreChallenge.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your Browser in `localhost:4200`
   ```
   http://localhost:4200  |  http://167.71.185.125
   ```
